# CampReady 🏕️

A South African camping and outdoor adventure planner Android app.
Plan your trip, pack smart, and stay safe in the wild.



## Features
- 📍 Browse 5 iconic SA camping locations with detailed info
- ⚡ Difficulty ratings — Easy, Moderate, Hard
- 📶 Signal availability per location
- 🔥 Fire rules per location
- ⚠️ Dangers and safety warnings
- 🚗 Distance to nearest town
- 🎒 Pre-loaded packing checklist with 20 essential items
- ✅ Check off items as you pack with strikethrough
- 💡 6 survival tips across categories — Water, Fire, Wildlife, Navigation, Weather
- 📱 Works fully offline — no internet required

## SA Locations Included
- 🏔️ Drakensberg Gardens — KwaZulu-Natal
- 🐧 Boulders Beach Camp — Western Cape
- 🦁 Kruger National Park — Limpopo/Mpumalanga
- 🪨 Cederberg Wilderness — Western Cape
- 🌿 Magoebaskloof — Limpopo

## Tech Stack
- Kotlin
- Android SDK
- Room Database (SQLite)
- Navigation Component
- View Binding
- Kotlin Coroutines & Flow
- ListAdapter with DiffCallback
- KSP (Kotlin Symbol Processing)
- Material Design Components

## How To Run
1. Clone the repo
2. Open in Android Studio
3. Run on emulator or physical device (API 26+)
4. App pre-populates with SA locations, packing items and survival tips on first launch
